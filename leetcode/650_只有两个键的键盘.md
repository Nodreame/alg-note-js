### 方法 3 素数分解

- 复杂度：
    - 时间 O(n^(1/2))
    - 空间 O(1)

``` js
var minSteps = function(n) {
    let res = 0
    let d = 2
    while (n > 1) {
        while (n % d === 0) {
            res += d
            n = Math.floor(n/d)
        }
        d++
    }
    return res
};
```
