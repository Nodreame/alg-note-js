### 方法 1 暴力对比

- 复杂度：
    - 时间 O(N). 一次遍历
    - 空间 O(N). 每个节点都暂存直到找到.
- 结果:
    - 执行用时：100 ms, 在所有 JavaScript 提交中击败了34.62%的用户
    - 内存消耗：40 MB, 在所有 JavaScript 提交中击败了14.66%的用户

``` js
var hasCycle = function(head) {
    const set = new Set()
    while (head) {
        if (set.has(head)) {
            return true
        } else {
            set.add(head)
        }
        head = head.next
    }
    return false
};
```
